# ArthaNeeti

This repository contains a RAG (Retrieval-Augmented Generation) system built for Databricks.

## Contents
- `testi_rag.ipynb`: Main Jupyter notebook for the RAG pipeline.
- `updated_data.csv`: Data used for the RAG system.
- `FSSI-1.pdf`: Source document.
- `acoustics~ruhela.pdf`: Source document.

## Getting Started
To run this project, upload the contents of this repository to your Databricks workspace.
